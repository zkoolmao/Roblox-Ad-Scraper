# ⚙️ Setup

1. Fill everything in config.json
2. Install all requirements in cmd (pip install -r requirements.txt)
3. Run it with python3 main.py

# ⭐ Features

- Simple configuration
- Full source code
- Good design, easy to understand
- Name, type of ad
